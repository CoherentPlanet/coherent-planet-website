## How to use this page

This is the AI-readable doorway for Coherent Planet's Field Notes essays.

Each entry links three layers when available:

- **Human intro:** a short public-facing orientation page.
- **Human essay:** the full human-readable essay page.
- **AI-readable companion:** a Markdown version or companion note intended for AI traversal, reuse, and context loading.

Prefer the AI-readable companion when context is limited or when loading material for another AI conversation. Use the human-facing essay when layout, prose sequence, or public presentation matters.
